# 設計書同期

> Input: なし（設計書と実コードの差分を自動検出）
> Output: 設計書の更新 + 実装との一致確認
> 所要時間: 1〜2時間
> 参照: `docs/00_設計成果物一覧.md`

---

## 何をするか

設計書（docs/）と実コード（app/, components/, lib/）を突き合わせて、ズレを全て直す。
設計図と実物が違うと、次に開発する人（AIも人間も）が間違った前提で作業してしまう。

---

## Phase 1: 差分検出

以下の3軸で設計書と実コードを突き合わせる。

### 軸1: DB設計（docs/12） vs 実テーブル

```bash
# マイグレーションファイルから実テーブル構造を取得
ls supabase/migrations/
# docs/12 に記載されたテーブルと突き合わせ
```

チェック:

- 設計書にあるが実装にないテーブル → 未実装として記録
- 実装にあるが設計書にないテーブル → 設計書に追記
- カラムの型・制約の不一致

### 軸2: API設計（docs/13） vs 実ルート

```bash
# 実APIルートを列挙
find app/api -name "route.ts" | sort
# docs/13 に記載されたエンドポイントと突き合わせ
```

チェック:

- エンドポイントの過不足
- HTTPメソッドの不一致
- リクエスト/レスポンスの型の不一致

### 軸3: 画面設計（docs/09） vs 実画面

```bash
# 実画面を列挙
find app -name "page.tsx" | sort
# docs/09 に記載された画面と突き合わせ
```

チェック:

- 画面の過不足
- 画面内の機能の不一致
- 画面遷移の不一致

## Phase 2: 差分リスト作成

| #   | 設計書  | 項目       | 状態        | 対応             |
| --- | ------- | ---------- | ----------- | ---------------- |
| 1   | docs/12 | テーブル名 | 設計 > 実装 | 未実装として注記 |
| 2   | docs/13 | /api/xxx   | 実装 > 設計 | 設計書に追記     |
| ... |         |            |             |                  |

### 状態の分類

| 状態            | 意味                       | 対応                                                |
| --------------- | -------------------------- | --------------------------------------------------- |
| **設計 = 実装** | 一致                       | 対応不要                                            |
| **設計 > 実装** | 設計書にあるが未実装       | 設計書に「未実装」注記。実装は build-feature の仕事 |
| **実装 > 設計** | 実装済みだが設計書に未反映 | **設計書を更新**                                    |
| **設計 ≠ 実装** | 両方あるが内容が違う       | **実装に合わせて設計書を更新**（実装が正）          |

## Phase 3: 設計書更新

「実装 > 設計」「設計 ≠ 実装」の項目を設計書に反映する。

### 更新ルール

1. **実装が正**: 実装と設計書が矛盾する場合、実装に合わせる（実装が動いている事実が最も信頼できる）
2. **フォーマット維持**: 既存の設計書フォーマットに合わせて更新
3. **差分最小**: 変更箇所のみ更新。関係ない部分は触らない
4. **日付記録**: 設計書の変更履歴セクションに更新日と変更内容を記録

## Phase 4: IMPLEMENTATION_STATUS 更新

`.agent/shared/IMPLEMENTATION_STATUS.md` の完了状況も、実コードの状態に合わせて更新する。

## Phase 5: 検証

```bash
# 設計書のリンク切れチェック
grep -rn "docs/" .agent/ --include="*.md" | while read line; do
  file=$(echo "$line" | grep -o "docs/[^ )]*")
  if [ ! -f "$file" ]; then
    echo "BROKEN: $line"
  fi
done
```
