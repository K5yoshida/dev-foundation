## 2026-03-04

### 対象: 定期チェック（v2: 指名検索CVR運用OS）

### 確信度: 18/25

### 判定: PIVOT

### WHO接続: 指名検索流入があり、1件のCV価値が高く、サイト改修が遅い中小事業者と、その導入を横展開する制作会社・コンサル・地域代理店の「既存サイトを変えずにCVを増やしたい」「短時間で複数顧客へ配りたい」という課題に直結している。

### WHAT接続: 「Capture / Convert / Prove / Improve / Scale」の5本柱のうち、現状は Capture と Convert の入口価値は成立しつつあるが、Prove（導入前後比較）、Improve（自動最適化）、Scale（パートナー大量展開）が弱く、価値提案の中心をまだ取り切れていない。

### 市場変化: KARTEは2025-12-01に広告連携コネクタを強化し、エンタープライズのデータ接続優位を継続。ReproのWeb SDKは2025-12〜2026-02に小規模な信頼性改善が続き、成熟した接客基盤として安定運用を強めている。一方でSTUDIOは2025-11-25にEditor 5.0を正式リリースし、AI機能とUI/速度改善でLP作成体験を前進。ペライチも2025-12-18に決済手数料改定を打ち出しており、汎用LP勢は「作る/売る」を強めている。したがって、特設ページ.comは汎用ビルダー競争ではなく「タグ1行導入 × 指名検索CVR特化 × 証明と横展開」にさらに寄せる必要がある。

### 人間アクション: 一次ターゲット（介護・建設・士業）を各1社ずつ短時間ヒアリングし、導入前後比較レポートと代理設置のどちらが継続判断に効くかを確認する。既存/想定パートナーに、テンプレ複製・月次レポート・代理設置のうち何が営業武器になるかを確認する。直近の無料登録→タグ設置→初回公開の離脱地点を実数で確認し、Wave 1の優先順位を「導入前後比較」「3CV自動選択」「設置完了支援」に寄せる。

## 2026-03-04（2回目 — Claude Code 定期チェック）

### 対象: v2 Wave 1（P0）全15機能 — 設計書5本を精読した上での再評価

### 確信度: 20/25

### 判定: GO

### WHO接続

レガシー産業の中小事業者（介護・建設・士業・飲食・美容、137.5万社）が抱える3つの課題を直接解決する:

1. **コスト障壁**: サイト改修100-300万円 → タグ1行で月額980円
2. **リテラシー障壁**: 85%がHTML理解なし → テンプレート選択5分で完結
3. **業者依存障壁**: 制作会社にロックイン → 自分で即設定・即効果確認

パートナーコンサルタント（制作会社・マーケ会社）の「クライアントにCVR向上を提案できる武器がない」も、テンプレートライブラリ + RS70% + パートナー管理で解決。

### WHAT接続

コア価値「指名検索で来た人を、1人も逃さない」を、3CVパターン統合（LP + モーダル直CV + LINE離脱キャプチャ）で実現。P0の15機能は「導入→設定→運用→証明」のジャーニー全体をカバー。競合比でコスト55倍差・時間100倍差は2026年3月時点でも有効。

### 市場変化

- **ペライチ**: アプリ対応・サブスク課金・出張撮影 → LP作成ツールの横展開方向。指名検索CVR特化とは別路線。脅威度低
- **KARTE**: ファネルレポート・広告媒体連携強化 → 分析深化方向。月10万円〜は変わらず。レガシー産業到達不可。脅威度低
- **Ptengine**: 動的コンテンツ対応強化 → ヒートマップ+A/Bテスト中心。LP+LINE+モーダル統合なし。脅威度低
- **新興**: 「指名検索CVR特化」の類似SaaSは検索ヒットなし → カテゴリ定義者の先行優位は有効
- **注目**: BtoB領域で「AIエージェント活用CVR改善」の言及が増加 → 将来的にAI CVR最適化（成長レバー4）の重要性が増す可能性
