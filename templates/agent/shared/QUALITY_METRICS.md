# Quality Metrics

## コード品質基準

| 指標                             | 基準           | ツール         |
| -------------------------------- | -------------- | -------------- |
| TypeScript型エラー               | 0件            | `tsc --noEmit` |
| ESLintエラー                     | 0件            | `eslint .`     |
| テストカバレッジ                 | 80%以上        | Vitest         |
| バンドルサイズ（管理画面）       | < 500KB (gzip) | next build     |
| バンドルサイズ（ウィジェットJS） | < 15KB (gzip)  | custom build   |

## パフォーマンス基準

| 指標                      | 基準    | 計測方法             |
| ------------------------- | ------- | -------------------- |
| ウィジェットJS読み込み    | < 100ms | Performance API      |
| LP表示 (LCP)              | < 2秒   | Lighthouse           |
| 管理画面初期表示 (LCP)    | < 3秒   | Lighthouse           |
| API応答時間 (p95)         | < 500ms | Vercel Analytics     |
| ウィジェットAPI応答 (p95) | < 100ms | CloudFlare Analytics |

## アクセシビリティ基準

| 指標                     | 基準           |
| ------------------------ | -------------- |
| Lighthouse Accessibility | 90以上         |
| キーボードナビゲーション | 全操作可能     |
| スクリーンリーダー       | 主要フロー対応 |
| カラーコントラスト       | WCAG AA準拠    |
