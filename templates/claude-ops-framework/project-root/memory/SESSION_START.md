## ステップ4: リスク分類 5 段階の再確認

本日予定している操作を 5 段階リスクで分類:

| Lv | ラベル | 承認 |
|----|--------|------|
| L0 | 🔴 IRREVERSIBLE | 対面協議 + 本人実行 |
| L1 | 🟠 HIGH RISK | 個別事前承認 + 24h滞留 |
| L2 | 🟡 MID RISK | Phase 着手時一括承認、事後報告 |
| L3 | 🟢 LOW | 事後報告のみ |

**分類が曖昧な場合は 1 段上を採用**。
