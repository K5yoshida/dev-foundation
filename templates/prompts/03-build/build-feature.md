# 機能実装

> Input: tasks.md（実装タスクリスト）
> Output: 実装コード + テスト通過
> 所要時間: 2〜4時間

---

## ルール

1. tasks.md の順番通りに実装
2. 各タスク完了後に `npm run build && npm run test` 確認
3. 完了したタスクは即座にチェック
4. 行き詰まったら力技禁止 → 再プラン

## 実装順序

```
DB層 → API層 → フロント層 → テスト → 検証
```

## 完了条件
- [ ] tasks.md の全タスクにチェック
- [ ] `npm run build` 通過
- [ ] `npm run test` 通過
- [ ] `npm run lint` 通過
