# テスト骨格作成（Phase 0）

> Input: spec.md + plan.md
> Output: テストファイル（.todo() テストケース）
> 所要時間: 30分〜1時間

---

## 何をするか

実装前にテストの「骨格」だけ作る。中身は空（.todo()）で、テストケース名だけ書く。
実装時にテストを埋めていくことで、テスト漏れを防ぐ。

## テンプレート

```typescript
describe('{{機能名}}', () => {
  describe('正常系', () => {
    it.todo('基本的な操作が成功する')
    it.todo('データが正しく保存される')
  })

  describe('異常系', () => {
    it.todo('不正な入力でエラーになる')
    it.todo('認証なしでアクセスできない')
  })

  describe('エッジケース', () => {
    it.todo('0件のとき空状態が表示される')
    it.todo('大量データでもパフォーマンスが維持される')
  })
})
```
