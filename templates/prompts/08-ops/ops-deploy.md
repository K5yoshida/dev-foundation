# デプロイチェックリスト

> Input: なし
> Output: デプロイ可否判定
> 所要時間: 30分〜1時間

---

## チェック

```bash
npm run build
npm run typecheck
npm run lint
npm run test
```

1つでも失敗 → デプロイ禁止。

## 環境変数確認
- 本番に必要な環境変数が全て設定されているか

## マイグレーション確認
- 新しいDBマイグレーションがあれば内容をレビュー

## セキュリティ最終チェック
- 機密情報がコードに含まれていないか
- `npm audit --production`

## デプロイ後確認
- トップページ表示
- ログイン動作
- 主要API応答
