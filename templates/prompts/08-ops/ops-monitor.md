# 運用監視セットアップ

> Input: なし
> Output: 監視設定 + アラート設定
> 所要時間: 1〜2時間

---

## チェック項目

1. **ヘルスチェック**: `/api/health` エンドポイント
2. **エラー監視**: 構造化ログ、内部情報漏洩防止
3. **パフォーマンス**: Web Vitals（LCP < 2.5s, FID < 100ms, CLS < 0.1）
4. **ビジネスメトリクス**: KPIの計測が動いているか
5. **アラート**: 定期バッチ稼働確認、ダウンタイム通知

## 完了条件
- [ ] ヘルスチェックエンドポイント実装済み
- [ ] エラーログが統一パターン
- [ ] ビジネスメトリクスが計測できている
